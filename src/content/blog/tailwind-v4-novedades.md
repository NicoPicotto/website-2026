Tailwind v4 represents the biggest change since v2. The CSS engine is now browser-native and all configuration has moved into CSS itself.

## The highlights

- No `tailwind.config.js` — configuration lives in CSS via `@theme`
- First-class CSS variables
- Significantly faster build performance
